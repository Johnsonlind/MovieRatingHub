# 影视多平台评分系统 🎬

## 项目简介 📝

这是一个集成了多个影视平台评分信息的系统，让用户能够便捷地查看和收藏不同平台的影视作品评分。通过本平台，用户可以轻松获取豆瓣、IMDb、TMDB、Letterboxd、烂番茄、Metacritic、Trakt 等平台的评分数据，管理自己的收藏与清单，并导出评分卡与榜单图。

---

## 主要功能 ✨

- 🔍 **多平台评分聚合**：一部影视多站评分一览，支持电影与剧集
- 👤 **用户认证系统**：注册 / 登录、JWT、个人资料
- ❤️ **个性化收藏列表**：创建清单、添加/删除收藏、收藏他人清单
- 🤝 **用户关注功能**：关注用户、查看关注列表
- 📊 **评分数据可视化**：评分分布、趋势与榜单展示
- 🔄 **实时数据更新**：评分与榜单短期缓存，定时同步
- 🌐 **跨平台兼容**：响应式布局、PWA 支持
- 📤 **导出分享**：评分卡 PNG、榜单图 PNG

---

## 技术栈 🛠️

### 后端

- **FastAPI**：API 与路由
- **SQLAlchemy + Alembic**：ORM 与数据库迁移
- **Redis**：评分缓存、TMDB 代理缓存、榜单短期缓存
- **JWT 认证**：登录与鉴权
- **Python 3.x**（推荐 3.12+）
- **Uvicorn**：ASGI 服务

### 前端

- **TypeScript**
- **React 18**
- **Tailwind CSS**
- **Vite 5**
- **React Router**、**TanStack Query**、**Lenis** 平滑滚动、**PWA (Workbox)**

### 数据与部署

- **MySQL 8+**：主库
- **TMDB / Trakt**：影视信息与图片
- **Nginx**、**Cloudflare**：生产部署与加速

---

## 环境要求 📋

- **Python 3.8+**（推荐 3.12+）
- **Node.js 16+**（推荐 18+）
- **Redis 6+**

## 主要功能模块 🔧

### 用户系统

- 注册 / 登录
- JWT 鉴权与刷新
- 个人资料管理

### 收藏功能

- 创建收藏列表（清单）
- 添加 / 删除收藏
- 收藏列表分享与收藏他人清单

### 社交功能

- 关注其他用户
- 查看关注列表
- 关注用户动态（可选）

### 评分聚合

- 多平台数据整合（豆瓣、IMDb、TMDB、Letterboxd、烂番茄、Metacritic、Trakt）
- 短期缓存与实时评分更新
- 评分趋势与分布展示

### 榜单与导出

- 首页聚合榜、公开榜单列表与详情
- 榜单虚拟滚动（大量条目时）
- 评分卡 PNG、榜单图 PNG 导出（队列与缓存优化）

### 管理后台

- 榜单录入与 TMDB 搜索（防抖）
- 公开榜单同步与定时抓取任务
